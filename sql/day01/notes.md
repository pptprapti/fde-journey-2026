SQL Day 1

What is SQL? 
SQL stands for Structured query language

SQL is used to 

* Read Data
* Filter Data 
* Sort Data
* Update Data
* Analyze Data

Example table

id                      name                        salary
1                       Prapti                      50000
2                       Anita                       60000
3                       Rahul                       45000

SELECT Statement
* Show all columns: 
    - SELECT * FROM employees;
* Show only names:
    - SELECT name FROM employees;
* Show only salaries:
    - SELECT salary from employees;
